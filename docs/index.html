Quick Start

Running in 5 minutes

One Docker Compose file. No complex setup, no cloud dependency. Your data stays on your infrastructure.

1

Clone & configure

Grab the repo and copy the environment template. All defaults work for local development.

git clone heedback.git
cp .env.example .env
2

Launch everything

One command spins up the API, dashboard, portal, database, Redis, MinIO, and email service.

docker compose up -d
3

Embed the widget

Drop a single script tag into your app. Identify users, customize colors, and start collecting feedback.

<script src="..."
  data-org="my-org"
  data-color="#6366f1">
</script>

Built for teams who care about their infrastructure

A modern stack designed for reliability and simplicity. Multi-tenant by design, so one deployment serves all your organizations.

Multi-tenant isolation — organization_id on every row, no data leaks
Session-based auth — secure cookies for admins, magic links for end users
Async job queue — emails, notifications, and heavy work via Bull + Redis
S3-compatible storage — MinIO in dev, any S3 provider in production
i18n ready — translatable articles, changelogs, and email templates (EN/FR)

Free, forever. Open source.

MIT licensed. Inspect the code, fork it, extend it. No vendor lock-in, no surprise invoices, no data harvesting.
